server.js                              створюємо файл входу в программу

mpm init -y                            ініціюємо проєкт

npm i express cors cookie-parser       встановлюємо потрібні для старту ліби

npm i nodemon --save-dev               кидаємо в залежність нодемон,щоб проєкт слідкував за змінами самостійно

npm i dotenv                           для роботи з середовищем

npm i mongodb mongoose                 для роботи з монгодб

.env(файл середовища)                  налаштовуємо базу на Монго https://cloud.mongodb.com/ та наш .env

mongoose.connect                       підключаємо базу в server.js

app.use                                підключаємо мідлвари в server.js

npm i joi                              підключаємо джой для комфортної роботи зі схемами

models                                 створюємо моделі юзера та інших коллекцій для монгодб

router                                 створюємо роути,за якими будуть іти запити

controllers                            створюємо функцію-контроллер,яка буде викликатись по якомусь з роутів

services                               щоб не писати в котролерах забагото логіки створюєму сервіси і виносимо значну частину логіки в них

npm i jsonwebtoken                     для роботи з JWT(генераціі токенів за допомоги секрет ворд)

npm i bcrypt uuid                      для бікрипт хешування пасворду, а юід для генераціі лінки веріфікаціі імейлу 

dtos                                   створюємо класс ЮзерДто щоб передавати з юзеру окрему інформацію

npm i nodemailer                       встановлюємо для надсилання листів користувачу
